# SénSanté 🏥

Assistant de santé communautaire avec IA

## Description
SénSanté est une application web destinée aux agents de santé communautaire
au Sénégal (Dakar, Thiès, Saint-Louis...).

## Stack technique
- **Frontend** : Next.js 14 (App Router) + Tailwind CSS
- **Backend** : API Routes Next.js
- **ORM** : Prisma
- **Base de données** : PostgreSQL
- **Auth** : NextAuth.js
- **IA** : Groq API (Llama 3)

## Installation
```bash
git clone https://github.com/GB463/Sensant-.git
cd sensante
npm install
npm run dev
```
